Data Ingestion

This is the technology demonstration platform showing proof of concept examples of the identified core services.

The PoC is implemented as a number of highly scalable static web pages providing access to services hosted by either the platform itself or by external providers of specalist technical capabilities.

 To deliver trusted data, methods, services and training, the platform needs to be supported by a number of key core services and it is these which the PoC is intended to evaluate.

Core Service Value

By providing a means to allow work in the cloud the statasticians can work from anywhere with a reasonably modern device connected to the internet.

Implementation

The Kylo integrated data ingestion platform allows non-developers to import, validate and transform data from a wide range of sources, log its providence and repeat on a scheduled basis. It uses Apache NiFi which can be visually programmed as a key part of the toolset designed fgor business users. 

 


Example

The platform allows data sources to be defined, ingested, validated and stored. NiFi is used to implement many of these functions with the metadata and provenance of of the data being stored within the system for later review.
