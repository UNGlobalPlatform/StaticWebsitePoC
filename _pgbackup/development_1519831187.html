Colaborative development

This is the technology demonstration platform showing proof of concept examples of the identified core services.

The PoC is implemented as a number of highly scalable static web pages providing access to services hosted by either the platform itself or by external providers of specalist technical capabilities.

 The platform will allow teams to collaborate on creating and then publishing solutions to challenges which others can then review and reuse as appropriate.

Core Service Value

By providing a means to allow collaborative development in the cloud the statasticians can work from anywhere with a reasonably modern device connected to the internet.

Implementation

Tools to allow coding in a number of different languages have been evaluated 

Example

The anaconda cloud allows developers to collaborate on projects anywhere in the world using a number of languages.

Algorithmia allows you to run any model function or algorithm as a microservice.
