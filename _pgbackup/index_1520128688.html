UN Global Platform

This is the technology demonstration platform showing proof of concept examples of the identified core services.

 To provide access to trusted data, methods, partners and learning, the platform needs to be supported by a number of key core technical services and it is the feasability of implementing these which this PoC is intended to demonstrate.

Value

The platform is intended to provide access to data, methods, partners and tools to support the statistical community in being able to create better statistics for better outcomes.

Implementation

This PoC is implemented as a number of highly scalable static web pages providing access to services hosted by either the platform itself or by external providers of specalist technical capabilities.
